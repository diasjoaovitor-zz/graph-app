## Tecnologias 

* Node.js
* React.js
* AntV G6

## Como rodar o projeto

* Faça o clone do repositório

```bash
$ git clone https://github.com/diasjoaovitor/graph-app
```

* Inicie o backend, caso deseje fazer upload de arquivos

```bash
$ cd graph-app
$ yarn ou npm i
$ yarn start ou npm start
```

* Inicie o frontend

```bash
$ cd client
$ yarn ou npm i
$ yarn start ou npm start
```
